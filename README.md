# Portfólio Profissional

Bem-vindo ao repositório do meu portfólio pessoal! Desenvolvi este espaço para demonstrar minha trajetória, meus estudos aprofundados e exibir detalhes sobre os projetos que construo, conectando tanto minha experiência com **Back-end** quanto **Front-end**.

## Tecnologias Utilizadas

Este projeto minimalista, moderno e altamente otimizado foi construído do zero utilizando as tecnologias abaixo, sem depender de templates prontos:

- **[Next.js](https://nextjs.org)** / **React**: Para roteamento otimizado (App Router) e renderização veloz de Server Components.
- **TypeScript**: Garantindo uma base sólida de desenvolvimento livre de erros clássicos de inferência.
- **Vanilla CSS (CSS Modules)**: Uma prova de conceito que não dependemos sempre do Tailwind/Bootstrap para entregar um layout esteticamente limpo e complexo com glassmorphism, grids flexíveis e transições elegantes.
